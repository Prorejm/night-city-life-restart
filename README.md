# 夜之城人生重开模拟器 🏙️

赛博朋克2077世界观的人生重开模拟器。基于 lifeRestart 引擎改造。

## 玩法

在夜之城中度过一生，从出生到死亡，经历帮派冲突、企业阴谋、义体改造、网络入侵、超梦体验…

每一次人生都是一段独特的故事，你的选择决定了你的结局。

## 特性

- **完整的赛博朋克世界观** — 出身、帮派、企业、义体、网络、超梦
- **6大属性体系** — STYLE(街头声望) / TECH(技术) / CHROME(义体) / EDDIES(欧元) / HUMANITY(人性) / LIFE(生命)
- **80+天赋** — 出身天赋 / 帮派关联 / 企业关联 / 义体天赋 / 特殊天赋 / 流浪者天赋
- **200+事件** — 覆盖0-100岁，包含夜之城各区、各大势力、历史事件
- **10+种死亡方式** — 赛博精神病 / 帮派处刑 / 企业暗杀 / 超梦脑死 / Max-Tac击毙…
- **物品系统** — 普通至不朽品质的武器和义体，含神话传说级不朽装备
- **成就系统** — 解锁夜之城传奇成就，获得来生酒吧专属酒品配方
- **赛博朋克风格UI** — 故障效果 / 扫描线 / 霓虹HUD / 深色主题

## 开发

```bash
npm install
npm run dev    # 开发模式
npm run build  # 生产构建
```

## 技术栈

JavaScript (ESM) + jQuery + Webpack 5 + CSS3
